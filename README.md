# 👟 Shoes - Online Sneaker Store

A modern, fully responsive **online shoe store web application** built with React.  
Users can browse shoes, select sizes and colors, add items to cart, and complete authentication-based access.  
The project also includes a powerful **admin panel** for full product management.

---

## 🚀 Features

### 🛍️ User Side
- 🔐 User authentication (Login / Register)
- 📧 Email + password based login system
- 👟 Browse shoe products
- 📏 Select size and color options
- 🛒 Product interaction for shopping experience
- 📱 Fully responsive design (mobile / tablet / desktop)
- ⚡ Fast and dynamic UI updates

---

### 🧑‍💼 Admin Panel
- 🔑 Admin login system (separate access)
- ➕ Add new shoe products
- 🗑️ Delete existing products
- ✏️ Edit shoe details:
  - Sizes
  - Colors
  - Discount rates
  - Product properties
- 📊 Full product management dashboard
